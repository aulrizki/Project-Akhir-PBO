Program ini adalah program yang ditujukan untuk memudahkan konsumen dalam pemesanan tiket pesawat 
berbasis Java Swing dan Java OOP.


Run Program :
- $ git clone https://github.com/aulrizki/Project-Akhir-PBO.git
NetBeans :
- Open Project "File yang telah di clone"
- Klik kanan pada project lalu klik run
Pada IDE Lain :
- $ javac BeliDong.java
- $ java BeliDong

Alur Program :
Ketika user membuka program maka ia akan diberi 2 pilihan yaitu :
- Informasi Penerbangan 
- Pemesanan Tiket
- Namun user diharuskan untuk mengecek informasi penerbangan terlebih dahulu sebelum melakukan pemesanan tiket.
- Setelah user mengecek informasi penerbangan, lalu pilih pemesanan tiket.
- Pada menu pemesanan tiket user harus mengisi data nama, kelas penerbangan, maskapai, tujuan, dan jadwal.
- Setelah mengisi data maka akan tampil no penerbangan sesuai dengan data yang diisi, lalu tekan tombol next
  untuk melanjutkan ke menu pembayaran.
- Pada menu pembayaran tertera harga yang harus dibayar sesuai dengan kriteria tiket yang diinginkan,
  user diharuskan menginput nominal sesuai harga yang tertera.
- Jika user telah membayar maka akan tampil data tiket yang telah di pesan.
